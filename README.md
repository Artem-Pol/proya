# Expense Tracker

## Автор
Артём Половинкин

## Описание
Программа "Expense Tracker" предназначена для отслеживания личных расходов. Пользователи могут добавлять записи о своих расходах, указывать сумму, категорию (еда, транспорт, развлечения) и дату. Программа также предоставляет возможность подсчета общей суммы расходов за все время.
